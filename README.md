# Proyectos de la Facultad

Este repositorio contiene trabajos prácticos realizados en el contexto de la facultad, enfocados principalmente en análisis de datos.

## 📁 Proyectos

### 📊 EPH - Análisis de empleo en Argentina
Análisis de datos de la Encuesta Permanente de Hogares (EPH) para estudiar la relación entre variables como nivel educativo, edad y sexo con el tipo de empleo (formal/informal) y los ingresos.

📂 Carpeta: `analisis_eph_argentina`

---

### 🚲 EcoBici - Análisis de uso en Ciudad Universitaria
Análisis del uso del sistema EcoBici en Ciudad Universitaria, evaluando la disponibilidad de bicicletas y espacios a lo largo del tiempo.

📂 Carpeta: `análisis_ecobicis_ciudad_universitaria`

---

## 🛠️ Herramientas utilizadas
- Python
- Jupyter Notebook
- Pandas
- Matplotlib / Seaborn
