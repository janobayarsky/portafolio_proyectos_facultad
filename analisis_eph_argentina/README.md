# 📊 Análisis de empleo en Argentina (EPH)

## 📌 Descripción
En este trabajo se analizan datos de la Encuesta Permanente de Hogares (EPH) con el objetivo de estudiar cómo distintas variables influyen en el tipo de empleo (formal e informal) y en los ingresos.

## 🎯 Objetivos
- Analizar la relación entre nivel educativo y tipo de empleo
- Estudiar diferencias salariales
- Evaluar el impacto de variables como edad, sexo y región

## 📂 Datos
Se utiliza el dataset `usu_individual_T325`, que contiene información individual relevada en la EPH.

## 🛠️ Herramientas utilizadas
- Python
- Pandas
- Jupyter Notebook
- Matplotlib / Seaborn

## 📈 Contenido
El notebook incluye:
- Limpieza y procesamiento de datos
- Análisis exploratorio
- Visualización de resultados
- Conclusiones

## 📎 Archivos
- `analisis_eph_argentina.ipynb`: desarrollo completo del trabajo
- `enunciado_eph`: consigna del trabajo práctico

- `EPH_registro_3T2025.pdf`: información detallada de los campos de la base de datos
